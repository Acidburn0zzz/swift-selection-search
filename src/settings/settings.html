Main features
Manage browser search engines
Opens Firefox's search engines manager. To select which engines are shown in the popup panel, use the "Select from available search engines" button below.
Select engines for popup
Enable/disable the search engines that are displayed on the popup panel (the context menu still shows all of them).
Show icon to copy selection
(default: Don't show)
Show "Copy to clipboard" icon in the panel.
(default: Don't show)
Show icon to open selection as a link
(default: Don't show)
Show "Open as link" icon in the panel.
(default: Don't show)
Behaviour
Opening behaviour
(default: Auto)
When to open the popup panel.
(default: Auto)
Popup location
(default: At cursor location)
Where the popup should appear after selecting text.
(default: At cursor location)
Hide popup on page scroll
(default: On)
Hides the popup panel when the page moves.
(default: On)
Hide popup on search engine click
(default: On)
Hides the popup panel when a search engine is clicked.
(default: On)
Key combination to open popup
(default: accel-shift-space, which is Ctrl+Shift+Spacebar on Windows/Linux and Cmd+Shift+Spacebar on Mac)
Opens the popup panel if you have text selected.
Use at least one: shift, control, alt, meta (Mac Command), accel (Control on Windows/Linux, Command on Mac). Clear the text box to disable the hotkey.
(default: accel-shift-space, which is Ctrl+Shift+Spacebar on Windows/Linux and Cmd+Shift+Spacebar on Mac)
Key combination to toggle auto popup opening
(default: accel-shift-x)
Toggles "Popup opening behaviour" between Auto and Keyboard-only modes (if not already Off). Use this when the popup is getting in the way of your browsing or text writing.
(default: accel-shift-x)
Left mouse button behaviour
(default: Open in this tab)
What to do when a search engine is clicked with the left mouse button.
(default: Open in this tab)
Middle mouse button behaviour
(default: Open in new background tab)
What to do when a search engine is clicked with the middle mouse button.
(default: Open in new background tab)
Panel animation duration
(default: 200)
Duration of panel animation, in milliseconds. 0 ms makes panel show instantly.
(default: 200)
Auto copy to clipboard
(default: Off)
Automatically copy selected text to clipboard when popup panel opens.
(default: Off)
Appearance
Use single row
(default: On)
Places all search engines in a single row, instead of a grid.
(default: On)
Number of icons per row in grid
(default: 4)
Max number of search engines per row when in grid (i.e. if "Use single row" is off).
(default: 4)
Icon size
(default: 24)
Search engine icon size in pixels.
(default: 24)
Icon padding
(default: 2)
Search engine icon horizontal padding in pixels.
(default: 2)
Icon hover behaviour
(default: Highlight & Move)
What happens to hovered search engine icons.
(default: Highlight & Move)
Popup background color
(default: white, #FFFFFF)
Color for the popup panel background. Firefox 49 and newer supports transparency (8-digit #rrggbbaa hexadecimals).
(default: white, #FFFFFF)
Popup highlight color
(default: blue, #3366FF)
Color for the popup panel highlight. Firefox 49 and newer supports transparency (8-digit #rrggbbaa hexadecimals).
(default: blue, #3366FF)
Popup horizontal padding
(default: 3)
Popup panel left/right padding, in pixels.
(default: 3)
Popup vertical padding
(default: 1)
Popup panel top/bottom padding, in pixels.
(default: 1)
Popup horizontal offset
(default: 0)
Popup panel horizontal offset in pixels (positive values move right).
(default: 0)
Popup vertical offset
(default: 0)
Popup panel vertical offset in pixels (positive values move up).
(default: 0)
Context menu
Is the context menu enabled?
(default: On)
Enables a "Search with..." sub-menu with all search engines. Appears in the context menu when you right click a page with text selected.
(default: On)
Context menu item click behaviour
(default: Open in new background tab)
What to do when a search engine is clicked in the context menu.
(default: Open in new background tab)
Context menu engines filter
(default: All)
What engines to show on the context menu.
(default: All)
Reset
Reset all settings
Immediately resets all options to their defaults.
