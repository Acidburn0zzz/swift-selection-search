Hey! Sorry for bothering you. I'm Swift Selection Search, but friends call me SSS!

I'm a WebExtension now, so I work on Firefox 57+. Hooray!

A few things you should know:

  • Pre-WebExtensions settings are incompatible with my new version. :( Please check the SSS settings page and change anything you'd like.
  • In WebExtensions I can't read search engines from your browser, but if you select the file that contains these engines for me, I can add them. Try it out in my settings page!
  • Now you can also define your own search engines. (It was requested by many families!)
  • The settings for custom shortcut keys are absent until Firefox supports them for WebExtensions. Sorry!
